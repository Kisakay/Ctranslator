# Translator project

This is a small project I made for train myself in the C language. The goal is to understand better how HTTP request work and how to use external library like libcurl.

I try to make a very light program that can translate text using Google Translate unofficial API. It is not official and it can break at any moment but it is ok for learning.

I also learn how to handle memory in C, like malloc and realloc, and how to manage string safely without crash the program.

Another goal of this project is to understand how to parse simple JSON response without using big library. I just take the first string from the result because I want keep it simple.

I use this project to improve my skills in system programming and network programming. It help me to understand low level behavior and how data is transfer on internet.

This is not a production tool, just a learning exercise.

For now I focus on simplicity and learning step by step.
